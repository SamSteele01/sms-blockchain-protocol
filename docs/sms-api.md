# SMS API

Text messages are sent to the phone number for the server. The format is CSV.

### Tweet (MVP - Impact Track)

`tweet, message`

command, message

### Coincap (bounty)

`coincap, rates, bitcoin`

command, [rates, exchanges, assets], [bitcoin, ethereum, XRP, lightcoin, EOS, ...]
